디자인 변경 정리 (v3)

  • 테두리 전부 제거. 카드 외곽·역할 태그·표 셀 모두 line 없음. shadow와 배경색으로만 구분.
  • 역할 컬러 단일 회색 톤. 봉제·자수·나염·워싱·완성·검품 모두 같은 회색 칩(#f3f4f6 / #6b7280). 컬러 강조 X.
  • 선택 시 강조: 파란색 X → 옅은 회색(#f9fafb + 1.5px #d1d5db) box-shadow.
  • 헤더 정렬: grid-template-columns: 36px 64px 64px 84px 1fr 80px 88px. 컬럼마다 위에 작은 라벨(출고날짜/오더날짜/품번/아이템/출고수량/진행) → 정렬 명확.
  • 완료 색상: 초록도 채도 낮춤. 옅은 민트 배경(#f6fdf9) + 진한 글자(#047857).
  • 주요 액션: 결제 체크 = dark(#111827), 완료 = green. 명세서 = 회색 보조 버튼.
  • 하단 합산 바: 어두운 배경 → 화이트 배경 + drop shadow. 톤 통일.
  • 타이포: 자간 -0.2~-0.5px 미세 조정. antialiased.
