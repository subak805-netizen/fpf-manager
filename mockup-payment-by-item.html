📐 통합 명세서 디자인 포인트

  • 표 한 장에 여러 출고일·여러 아이템 다 합치기. 회차별로 소계 표시.
  • 품목 행: 첫 줄에 출고일·품목·컬러·수량·단가·금액. 같은 출고일·품목이면 컬러만 다음 줄.
  • 소계 행: 회색 배경으로 그 회차/그룹 합계 명확히.
  • 합계 영역: 공급가액 → 부가세 → 최종 합계(VAT 포함 강조).
  • 입금계좌: 하단에 청록색 박스.
  • 박스비/택배비 같은 fees는 별도 행으로 노출 (단가×수량 명시).
  • 액션 버튼: 텍스트 복사 / 카카오톡 / 이미지 저장 / 닫기.

📐 디자인 정리

  • 아이템 = 외곽 큰 카드. 헤더에 품번·이름·브랜드·컬러·총 수량·진행 상태(N/M 완료)
  • 출고일별 분리. 한 아이템이 5/2 + 5/15에 따로 출고됐다면 안에 두 행. 회차 표시
  • 공장 카드 = 195px 균일 그리드. 역할 태그(컬러로 봉제·자수·나염·워싱·완성·검품 구분), 공장명, 단가·수량 분해, 청구액, 명세서/체크 버튼
  • 통합 명세서. 같은 공장에 여러 아이템·회차가 있으면 한 장의 명세서로 합쳐서 받기
