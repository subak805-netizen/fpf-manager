① Supabase 계정 만들기
→ supabase.com 접속 → Start your project → GitHub 또는 이메일로 가입

② 새 프로젝트 만들기
→ New project → 이름 아무거나 입력 → 비밀번호 설정 → Northeast Asia 선택 → Create
→ 프로젝트 생성까지 1~2분 기다리기

③ 테이블 만들기 (SQL 실행)
→ 왼쪽 메뉴 SQL Editor → New query
→ 아래 SQL 전체 복사 → 붙여넣기 → Run (Ctrl+Enter)
create table if not exists fpm_data (
  key text primary key,
  value text,
  ts timestamptz default now()
);
alter table fpm_data enable row level security;
create policy "allow_all" on fpm_data for all using (true) with check (true);

→ anon · public 키 복사 → 아래 두 번째 칸에 붙여넣기
