🛠 fpf-manager 워크플로 개선안 시안

코드 반영 전 미리보기예요. 3개 핵심 변경안 — 각각 보시고 좋다/아니다/이렇게 바꿔달라 알려주세요.

시안 1

다음 단계 CTA — "다음에 뭐하지?" 안내

화면마다 다음 액션을 알려주는 초록 박스를 자동 노출. 사용자가 흐름 따라가는 부담 줄임.

현재 (Before)
아이템 등록 폼
이름·품번·컬러·사이즈·원단·부자재·비용·공장 입력
저장
저장하면 토스트 한 번 뜨고 끝. 다음에 뭐해야 할지 안내 없음.
사용자가 매번 "샘플 만들어야 하나? 메인 발주 만들어야 하나?" 판단해야 함.
개선 (After)
아이템 등록 폼
이름·품번·컬러·사이즈·원단·부자재·비용·공장 입력
저장
✅ 아이템 저장 완료!
다음 단계를 골라주세요:
🪡 샘플 만들기
🏭 메인 발주 만들기
💰 원가 보기
저장 후 다음 액션 버튼이 자동 등장. 클릭하면 그 화면으로 점프 + 아이템 정보 자동 연결.

적용 위치 (워크플로 전체)

메인 발주 생성 후
→ 📋 발주서 작성 / 📦 수량 입력

전체 입고 후

전체 출고 후

시안 2

오더 상세 — 진행 단계 표시 바

오더 상단에 작은 진행 바. 한 눈에 어디까지 했는지 보임. 클릭하면 해당 서브탭으로 점프.

서브탭만 가로로 6개. 어디까지 했는지 안 보임.

시안 3

아이템 폼 — 섹션 토글 (접기/펼치기)

긴 폼을 섹션별로 접어서 스크롤 부담 줄임. 처음엔 기본정보만 펼침. 다른 섹션은 필요할 때 펼침.

현재 (Before) — 모든 섹션 항상 펼침
▼ 기본 정보
이름, 품번, 브랜드, 시즌, 클라이언트
▼ 컬러·사이즈
컬러 3개, 사이즈 4개
▼ 가공비·공장 (6개)
공임 25000원/아이케이, 완성비 4000원/다산사, 검품 0/블루패킹, ...
▼ 원단
새틴 60수, 안감 폴리...
▼ 부자재
단추 15mm 흰색 4구, 지퍼 3호 12" 오픈, 케어라벨, 메인라벨, ...
▼ 패턴·샘플·1회
패턴비 80000원, 샘플비 25000원, 1회비 10000원
스크롤이 길어짐. 부자재가 10개 넘어가면 끝없이 내려야 함.
개선 (After) — 섹션 토글
▼ 기본 정보 ✓ 입력됨
이름·품번·브랜드·시즌·클라이언트 입력 영역...
▶ 컬러·사이즈 ✓ 컬러 3 · 사이즈 4
▶ 가공비·공장 ✓ 6개
▶ 원단 ✓ 2개
▶ 부자재 ✓ 12개
▶ 패턴·샘플·1회 미입력
처음엔 기본정보만 펼침. 다른 섹션은 클릭해서 펼침. 각 섹션 옆에 입력 상태 미리보기 (예: "원단 2개").
시안 4 (보너스)

결제 관리 — 거래명세서 자동 펼침 제거

현재 공장 카드 안에 거래명세서가 자동으로 펼쳐져서 화면이 매우 길어요. 버튼 클릭으로 모달 열기.

공장 카드 안에 명세서 자동 펼침 → 화면 끝없이 길어짐.

🎯 종합 추천

위 4가지 중에서 임팩트 큰 순서로:

  1. 시안 1 (다음 단계 CTA) — 사용자가 매번 "다음 뭐하지?" 안 헤매도 됨. 가장 큰 변화
  2. 시안 2 (진행 단계 바) — 한 눈에 어디 막혔는지 보임
  3. 시안 4 (명세서 모달화) — 빠른 fix. 화면 길이 줄임
  4. 시안 3 (폼 토글) — 입력 부담 큰 사용자에게 도움. 위 3개보다는 후순위

📌 한 번에 다 하지 말고 1번부터 차근차근. 변경마다 사용자께서 직접 사용해보시고 피드백 주시는 게 안전해요.
