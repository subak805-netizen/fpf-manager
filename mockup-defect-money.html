2부. 비품 출고 연결 — A안 / B안 중 골라주세요
A안. 대기 배너 + 불일치 경고 추천
1
검품에서 비품 6장 발생 → 불량관리에 "비품 전환 6장"으로 등록 (사진·유형 그대로)
2
나중에 출고 입력창을 열면 배너가 먼저 보여요:
비품 6장 대기 중 — 불량관리에서 비품 전환 6장 등록됨.
이 수량은 [비품 출고]를 켜고 따로 입력하세요. (공임 50% 자동)
3
등록된 비품(6장)과 실제 비품 출고 수량이 안 맞으면 결제탭 카드에 경고:
비품 6장 등록 · 출고기록 0장 — 확인 필요
장점 수량은 항상 사장님이 직접 입력 — 자동으로 잘못 들어갈 일 없음. 깜빡해도 경고가 결제 전에 잡아줌
단점 출고 때 비품 입력을 한 번은 직접 해야 함
B안. 자동 회차 생성
1
불량관리에서 "비품 전환 6장" 등록하면서 컬러·날짜 입력
2
그 순간 출고 기록에 비품 회차 6장이 자동으로 추가됨 (공임 50% 자동)
3
출고 입력창에는 이미 들어가 있으니 손댈 것 없음
장점 손이 제일 적게 감
단점 불량관리 입력 실수가 출고수량·결제에 바로 들어감. 실제 출고일과 등록일이 다르면 날짜 어긋남. 검품소 수량과 이중으로 세다 꼬일 수 있음
정해줄 것
1) 비품 연결: A안 / B안 어느 쪽으로 할까요?
2) A안이면 배너 문구·경고 위치 이대로 괜찮은지
3) 1부(공제 연결)는 이미 적용됐어요 — 새로고침 후 결제탭에서 실제로 확인해보세요
