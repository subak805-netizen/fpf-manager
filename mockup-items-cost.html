📐 디자인 핵심

  • 좌측 리스트: 320px 폭, 검색·필터·아이템 카드. 카드에 장당 원가 미리보기.
  • 우측 원가계산서: 선택된 아이템의 원가. 헤더(브랜드·품번·아이템명) + 4섹션(원단·부자재·가공비·기타) + 합계.
  • 인라인 편집: 모든 값이 input. 클릭하면 바로 수정. 포커스 아웃 시 자동 저장.
  • 탭 이동 X: 아이템 선택과 원가 수정이 한 화면. 작업 흐름 빠름.
  • 장당 원가 미리보기: 좌측 카드에도 미리 표시 → 비교 빠름.
  • 합계 자동: 원단·부자재·가공비·기타 → 장당 원가 + 총 원가.

🤔 동작 흐름 (비교)

  • 기존: 아이템 탭 → 원하는 아이템 클릭 → 편집 모달 → 비용 입력 → 닫기 → 원가계산서 탭으로 이동 → 그 아이템 검색 → 확인
  • 변경 후: 아이템 탭에서 좌측 아이템 클릭 → 우측에 원가 즉시 표시 → 인라인 수정 → 끝
  • 탭 이동·검색 반복 사라짐. 작업 속도 2~3배 빠름.
