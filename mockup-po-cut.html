① 오더 → 재단 → 출고 (재단 추가)

맨 위 흐름에 재단을 끼우고, 표에도 "재단" 칸 추가. 재단이 오더보다 많으면 강조

② 재단 추가분 자동 입력

추가발주 → "재단 추가분" 누르면, 재단−오더가 컬러별로 자동 채워짐 (수정 가능)

추가 발주 — 에코백
수량 추가분재단 추가분불량 분
재단한 수량이 오더보다 많은 만큼 자동 입력됐어요. (재단 여유분·보충)
컬러오더재단추가분(자동)
노랑101515−10 자동
소라202525−20 자동
네이비557575−55 자동
추가 합계: 30장
재단을 더 했으면 그만큼 원단·부자재도 더 필요하니, 추가분을 자동으로 채워줘요. 숫자는 직접 고칠 수 있어요.

작동 방식: 공정별 "재단" 수량(컬러·사이즈별 저장됨)에서 오더 수량을 빼서 추가분으로 자동 입력. 재단을 아직 안 했거나 오더와 같으면 추가분 0(자동입력 안 함). 직접 수정도 가능.
