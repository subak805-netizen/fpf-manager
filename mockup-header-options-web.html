A vs C (웹 기준)
A 풀 슬림 — 상단 1줄, 작업창 가장 넓음. 탭 바꿀 때 ☰ 한 번 더 눌러야 함. 폰·웹 동일 동작이라 일관됨.
C 사이드바 — 왼쪽에 탭이 항상 보여 이동이 한 번에(웹에선 메뉴 누르려 펼칠 필요 없음). 넓은 모니터에 잘 맞음. 단 좌측 200px를 늘 차지. 폰에선 같은 C가 하단 탭바로 바뀜.
👉 C면 "웹=사이드바 / 폰=하단탭바"로 화면 크기에 따라 자동 전환 — 보통 가장 익숙한 앱 구조예요.
