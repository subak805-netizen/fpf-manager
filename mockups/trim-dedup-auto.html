중복 부자재 자동정리 〔제안〕
띄어쓰기·철자만 다른 부자재(코메즈밴드 ↔ 코메즈 밴드)를 자동으로 한 그룹으로 묶어서, 그룹당 한 번에 합쳐요. 대표 하나만 고르면 나머지가 흡수되고 그걸 쓰는 아이템도 자동으로 옮겨집니다.
지금은: 자재 행 찾기 → 「병합」 → 모달에서 상대 검색·선택 → 실행 을 중복 한 쌍마다 반복. 게다가 "중복 줄 찾기"는 띄어쓰기 변형을 못 잡아요.

규격(호수)이 달라서 자동합치기 대상은 아님 — 정말 같은 거면 수동으로 합치세요.
이렇게 돼요.
① 띄어쓰기·철자 변형을 자동 그룹핑(이름 공백·대소문자 무시) → 못 찾던 중복도 다 보임
② 그룹마다 대표 하나 선택(기본=제일 많이 쓰이고 단가 있는 것) → 「합치기」 한 번에 흡수+아이템 재연결
③ 규격·색상이 다르면 자동 대상에서 빼고 "확인필요"로 표시(실수 방지)
④ 이름·단가 완전히 같은 안전한 건 「한 번에 합치기」로 일괄
안전장치: 기존 병합 로직 그대로 사용(아이템 자동이동·단가보존·되돌리기 지원). 합치기 전 단가장 자동백업.

※ 정할 것: 자동 그룹 기준을 같은 거래처 안에서만 볼까요, 아니면 거래처가 달라도 이름 같으면 묶어 볼까요? (기본은 같은 거래처 안 — 더 안전)
