로스 프리셋 — 간단 추가·수정 〔제안〕
지금은 프리셋 추가가 물음창 4번(이름 → "pct/set/add" 직접 타이핑 → 값 → "all/fab/trim..." 타이핑)이라 헷갈리고, 수정 기능이 아예 없어요(추가·삭제만).
제안: 프리셋마다 「수정」 버튼 + 누르면 한 카드 안에서 이름·방식 토글·값·대상칩으로 바로 만들고 고치기.
제안 — 프리셋에 「수정」 + 인라인 에디터
로스 / 여분 자재별 % · 벌치 · +개 (이 발주서에만)
프리셋 기본 5% 라벨 +10개 단추 1벌치 ＋ 프리셋
칩을 누르면 적용·수정·삭제를 한 곳에서
라벨 +10개
개
전체원단부자재 라벨단추·스냅지퍼
이 발주서의 모든 라벨에 +10개 적용
· 방식은 타이핑 대신 [%｜벌치｜+개] 토글 (로스 패널이랑 똑같은 버튼).
· 대상도 칩으로 한 번 탭 (전체/원단/부자재/라벨/단추·스냅/지퍼).
· 수정 누르면 그 프리셋 값이 그대로 채워져서 바로 고침. +프리셋은 빈 칸으로 열림.
